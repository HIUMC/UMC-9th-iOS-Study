## 📑 커밋 컨벤션
### 🏷️ 커밋 태그 가이드

 | 태그        | 설명                                                   |
|-------------|--------------------------------------------------------|
| feat:      | 새로운 기능 추가                                       |
| fix:       | 버그 수정                                              |
| refactor:  | 코드 리팩토링 (기능 변경 없이 구조 개선)              |
| style:     | 코드 포맷팅, 세미콜론 누락, 들여쓰기 수정 등          |
| docs:      | README, 문서 수정                                     |
| test:     | 테스트 코드 추가 및 수정                              |
| chore:    | 패키지 매니저 설정, 빌드 설정 등 기타 작업           |
| design:    | UI, CSS, 레이아웃 등 디자인 관련 수정                |
| hotfix:    | 운영 중 긴급 수정이 필요한 버그 대응                 |
| CI/CD:     | 배포 관련 설정, 워크플로우 구성 등                    |

### ✅ 커밋 예시 모음
